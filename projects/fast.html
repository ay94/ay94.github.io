Multilingual NER & MT Benchmarking Framework

Incomplete — methodology fully documented · 2023

The Project

Context & Objectives

The most ambitious NLP project in scope across CASM's portfolio. The goal was to systematically identify and rank the best performing NER models and machine translation approaches for 26 languages, including several low-resource African languages rarely covered in NLP benchmarking literature.

The project was stopped before all 26 languages were completed, but the framework and methodology are fully documented and transferable.

My Role

Contributions

Technical Lead. I designed the benchmarking framework, built the evaluation infrastructure, and completed the German language evaluation as a reference implementation.

I also wrote the NER Evaluation Workflow document and the MT Evaluation Framework — both documented as reusable internal methodologies and applied in subsequent projects.

Method

Benchmarking Framework

NER Benchmarking (per language):

  1. Model identification — HuggingFace model hub and leaderboards (BERT LANG STREET) searched for candidate models per language
  2. Benchmark dataset identification — language-specific benchmarks sourced (CoNLL, xtreme/PAN-X, GermEval14 for German, etc.)
  3. Benchmark evaluation — each model evaluated using seqeval (entity-level F1) and sklearn (boundary-level flat F1) across PER, LOC, ORG, MISC
  4. Model ranking — models ranked per entity type and overall
  5. Project-specific validation — top-ranked models applied to project data, with analyst review of sample outputs

MT Benchmarking:

  • Comparative analysis of BLEU, ROUGE, METEOR, TER and BERTScore
  • BERTScore selected as primary metric for embedding-based downstream tasks
  • Error analysis: named entity preservation, tokenisation sequence length, information loss

Outcomes

Results & Impact

Full benchmarking framework designed and documented. German language evaluation completed as reference implementation (3 models × 2 benchmarks).

NER Evaluation Workflow and MT Evaluation Framework written as reusable internal methodology documents — both applied in subsequent projects. Project stopped before all 26 languages completed; framework fully transferable.

Languages scoped: Afrikaans, Arabic, Bulgarian, Czech, German, Greek, English, Spanish, Farsi, French, Hausa, Hindi, Indonesian, Japanese, Malay, Portuguese, Romanian, Russian, Slovak, Serbo-Croatian, Swahili, Thai, Turkish, Urdu, Xhosa, Chinese (Mandarin), Zulu
