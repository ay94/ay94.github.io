Sweden's Image Under Disinformation: Multilingual Analysis

Completed · 2023

The Project

Context & Objectives

The Swedish Institute commissioned CASM Technology to analyse how two major international incidents were reshaping Sweden's global image — the Quran Burning protests and the LVU child welfare law controversy. Both incidents triggered significant disinformation campaigns across Muslim-majority countries, damaging Sweden's diplomatic relationships and threatening its international standing.

The project required analysing social media and mainstream media content across six language communities simultaneously: Arabic, English, Urdu, Farsi, Turkish and Indonesian.

My Role

Contributions

Technical Lead. I designed and built the full multilingual analysis pipeline — including a machine translation layer, a multilayered topic modelling framework, and separate analysis streams for each narrative.

I also wrote the machine translation evaluation framework used to validate translation quality before analysis — now documented as a reusable internal methodology.

Method

Approach & Pipeline

  1. Data collection — social media and mainstream media across 6 languages: Arabic, English, Urdu, Farsi, Turkish, Indonesian
  2. Machine translation — all non-English content translated to English; quality validated using BERTScore as primary metric
  3. Preprocessing — URL/hashtag/mention removal, deduplication, language filtering
  4. Layer 1 topic model — generic BERTopic across full dataset → 5 broad themes
  5. Layer 2 topic models — per-theme BERTopic models → granular sub-topics within each theme, refined with SetFit few-shot classifiers
  6. Layer 3 (LVU) — deep topic model for highly granular analysis of LVU narratives specifically
  7. Three narrative streams analysed separately and comparatively: Quran Burning, LVU, Overlapping Narratives

Outcomes

Results & Impact

Full multilingual narrative landscape mapped across 6 language communities. Three narrative streams analysed separately and comparatively.

Multilayered topic modelling framework developed — reused in subsequent projects (Ofcom). Machine translation evaluation framework documented as reusable internal methodology.
